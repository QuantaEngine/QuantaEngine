# Roadmap

## v0.1: seed engine

- GitHub-ready package layout.
- Reproducible config-driven toy universe generation.
- Scalar-field lattice, expansion background, CLI, tests, examples.

## v0.2: law specification

- Add a `LawBook` or `Action` abstraction.
- Represent fields, symmetries, couplings, potentials, and particles in a structured way.
- Support JSON/YAML law files.

## v0.3: HEP-flavored modules

- Effective Lagrangian interface.
- Gauge-field prototype.
- Spontaneous symmetry breaking toy models.
- Phase-transition and bubble-nucleation-inspired modules.

## v0.4: cosmic structure

- Particle-mesh gravity.
- Density perturbation growth.
- Simple halo/catalog generation.
- Synthetic observables.

## v0.5: acceleration and scans

- Optional JAX/CuPy/PyTorch backends.
- Large parameter scans.
- AI-assisted search over microscopic laws.

## v1.0: validated engine core

- Stable public APIs.
- Benchmarks.
- Physics validation suite.
- Documentation for external contributors.
